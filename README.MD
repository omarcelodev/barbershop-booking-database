# 💈 Barbershop DB — Sistema de Agendamentos para Barbearia

Projeto de modelagem e documentação de banco de dados desenvolvido na disciplina de Banco de Dados — UniGoiás (2025).

---

# 📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de centralizar e automatizar o processo de agendamentos de uma barbearia, integrando clientes, barbeiros, serviços e horários em um único ambiente digital.

Além do gerenciamento de agendamentos, o sistema também prevê notificações automáticas de lembrete com base no status de cada atendimento.

O projeto envolve:

* Modelagem de banco de dados relacional
* Estruturação e manipulação em SQL
* Definição de regras de negócio
* Organização de dados para futuras análises e integração com BI

---

# 🚨 Problema Resolvido

Barbearias que realizam agendamentos manualmente estão sujeitas a diversos problemas, como:

* Sobreposição de horários
* Esquecimento de atendimentos
* Falta de controle sobre a agenda dos barbeiros
* Dificuldade no gerenciamento de serviços e clientes

O sistema resolve essas questões centralizando os agendamentos em uma única plataforma, promovendo maior organização, consistência e eficiência operacional.

---

# 🧩 Entidades do Sistema

O banco de dados foi estruturado com as seguintes entidades:

* Usuario
* Barbeiro
* Servico
* Agenda
* Agendamento
* Notificacao

# 📊 Dashboard e Análise de Dados

Embora não fosse um requisito obrigatório da atividade, também foi desenvolvido um dashboard no Power BI para expandir o projeto para a camada analítica.

O dashboard apresenta métricas como:

* Total de agendamentos
* Clientes únicos
* Serviços mais realizados
* Indicadores de faturamento
* Fluxo de atendimentos
